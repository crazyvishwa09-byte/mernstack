SELECTORS

TYPES:

  1. SIMPLE SELECTORS

  • CLASS
  • ID
  • UNIVERSAL
  • GROUP
  • ELEMENT

CLASS

->In CSS, a class selector is used to select and apply styles to HTML elements that have a specific class attribute value.

->Unlike ID selectors, which are intended for unique elements, class selectors can be applied to multiple elements, allowing for consistent styling across various parts of a webpage.

ID

->The ID selector in CSS is used to select a single, unique HTML element within a document by referencing its id attribute.

->The ID selector is represented by a hash symbol (#) followed by the unique id value of the HTML element you want to target.

UNIVERSAL

The universal selector in CSS is represented by an asterisk (*). It is a special type of selector that targets and applies styles to every single element within the document tree of an HTML page.

GROUP

The grouping selector in CSS allows the application of the same style rules to multiple HTML elements by listing the selectors separated by commas.

This technique is used to minimize code repetition and improve the readability and maintainability of CSS stylesheets.

ELEMENT

The element selector in CSS targets HTML elements based on their tag name. This selector applies styles to all instances of a specific HTML element throughout a document.
